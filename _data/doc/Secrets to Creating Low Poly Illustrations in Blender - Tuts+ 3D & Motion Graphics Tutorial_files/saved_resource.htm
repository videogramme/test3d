                                                                                                                        bergholt • a month ago

                                                                                                                        And for those that use 3dstudiomax,
                                                                                                                        (I have posted this on Abduzeedo months ago http://abduzeedo.com/mesmerizi... )
                                                                                                                        check this out:
                                                                                                                        In 3DSmax:
                                                                                                                        1) make an object, or import a high poly 3d model,
                                                                                                                        2) convert it to Poly,
                                                                                                                        3) apply Optimize modifier from roll-out ( will lower the poly count), I recommend: face thres:10, edge thres:1.0, Bias: 0.02, the rest leave them as it is, experiment as you like though!
                                                                                                                        4) change the viewport to facetes, so you can see the triangles in your viewport,
                                                                                                                        5) press (M) and tick "facetes". This tick will render the facetes you see in the viewport, if you don't you will only render the object smoothed, just with less polygons!
                                                                                                                        6) appy noise maps or anything else you want for material to look, e.g. like paper
                                                                                                                        7) add sunlight system and render it with renderer you like, mentalray, v-ray, etc.
                                                                                                                        8) voila!!! you 've got it!
                                                                                                                        Sooo simple!!
